# How to run the web
- Install XAMPP
- Find and open 'httpd.conf' file in XAMPP setting folder (default: C:/xampp/apache/conf)
- Find 'DocumentRoot' section in file, change the default path into the new path link to this bookstore folder
- Start Apache and MySQL in XAMPP
- Open your brower with path: localhost/bookstore

# How to setup database
- Open browser with path: localhost/phpmyadmin
- Import database file ('bookstore.sql') which is included in above folder
